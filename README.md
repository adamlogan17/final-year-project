# svs-paediatric-delerium 
## How to start the project
### Frontend 
* ```cd example-frontend```
* ```npm run start```

Before the first time you run the program you need to use the command ```npm install```
### Backend
* ```cd APIs\example-api```
* ```npm run start:dev```

Before the first time you run the program you need to use the command ```npm install```
